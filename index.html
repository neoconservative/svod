How it works

Projects of providing our services represent the complete modelling process. This process is closely modelled on the industry standard called Cross-Industry Standard Process for Data Mining (CRISP-DM).

We adjust processes by our experiences and particular customer circumstances. We use a wide range of the latest computer science achievements to make this process as efficient and accurate as possible.

We use CRISP-DM as a formal and transparent methodology to go with our customers through the entire process from start to finish. This way we monitor progress and are in control of important decisions at each project stage together with our customer.

Use cases

Improve user experience

We can analyse customer journeys through all interaction
points and make a prediction what should be the next action
with particular customer to improve their experience.
Managing customers interaction this way allow you Increase
NPS and prevent customer’s churn.

Tailored recommended system

We can make recommended system that properly fits your
customers. Your business will increase customer lifetime value
with cross-sell and upsell proposals personalization. It also
can work in call centers and offline shops.

Let customers speak to the best agent

Given the data of previous customer’s behaviour and agent’s
track record. We can predict the best agent for particular
customer’s call. Choosing the best agent will increase sells as
well as improve customer satisfaction.

Technologies we use

Descriptive Analysis - describe data and facts
through as its representations, e.g. Data
Visualization, BI.

Estimation - estimate values of a continuous
variable by many predictor variables, e.g.
Regression Analysis, Neural Networks.

Forecasting - forecast values for a phenomenon
ahead of time – e.g. Regression Analysis, Time
Series.

Classification - supervised classification of a
discrete variable by many predictor variables –
e.g. Logistic Regression, Support Vector
Machines and Decision Trees.

Clustering - unsupervised classification
of variables into fixed classes, e.g. K-mean
clusters, Factor Analysis.

Association - establish association between
objects using lead probabilities, e.g. Market
Baskets, Recommender Systems.
